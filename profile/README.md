# Diefi Predictive Maintenance Data Logger System 🚀

Welcome to the **G-DIEFI** organization. This system is designed for high-frequency vibration analysis and real-time sensor monitoring.

### 🛠 Core Components
* **[pm_backend](./pm_backend):** Python-based engine handling MongoDB ingestion and Socket streams.
* **[pm_sensor](./pm_sensor):** Firmware (C) for the data acquisition units.
* **Frontend:** Real-time FFT visualization and device management dashboard.
